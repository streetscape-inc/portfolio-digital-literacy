# portfolio-digital-literacy
Digital Literacy Portfolio

📚 Digital Literacy Project

📌 Introduction

This project focuses on digital literacy, which refers to the ability to use digital tools, platforms, and technologies effectively and responsibly. It highlights the importance of digital skills in education, communication, and online safety.

🎯 Objectives

* To understand the concept of digital literacy
* To explore various digital platforms
* To develop awareness about online safety and cybercrime
* To improve communication through digital tools

📂 Project Structure

📊 Task 1: Presentation

A presentation on digital literacy created using Canva, explaining its meaning, importance, components, and challenges.

🧩 Task 2: Digital Portfolio

Creation of digital profiles (LinkedIn and Canva) along with reflection notes on managing online identity.

💻 Task 3: Platforms and Activities

Screenshots and activity proof of using platforms like GitHub and Canva, along with notes on their usage.

✉️ Task 4: Email Etiquette

Sample professional email drafts and a checklist highlighting proper email communication practices.

🛡️ Task 5: Cybercrime Awareness

A case study on phishing attacks along with a checklist of preventive measures to ensure online safety.

🛠️ Tools Used

* GitHub
* Canva
  
📚 Module-wise Summary

📊 Module 1: Hands On Training On Presentation Software

This module focused on developing practical skills in using presentation software such as Canva. It involved creating structured and visually appealing presentations, understanding slide design, and effectively organizing information for better communication.

🧩 Module 2: Building a Digital Portfolio

This module emphasized the creation of a digital portfolio by setting up profiles on platforms like LinkedIn and Canva. It helped in understanding the importance of maintaining a professional online identity and managing personal information responsibly.

💻 Module 3: Programming Platforms & Collaborative Learning Platforms

This module introduced platforms like GitHub that support collaboration and project management. It provided hands-on experience in organizing files, sharing work, and understanding the basics of version control and teamwork in a digital environment.

✉️ Module 4: Email Etiquette & The Basics Of Social Media Etiquette

This module covered the principles of professional communication through emails and social media. It included proper email formatting, polite language, and guidelines for maintaining respectful and responsible behavior on online platforms.

🛡️ Module 5: Awareness On Cyber Crime And Financial Fraudulent

This module focused on creating awareness about cybercrime and financial frauds such as phishing and online scams. It highlighted the importance of cybersecurity practices, protecting personal data, and staying cautious while using digital services.

📌 Conclusion

This project helped in developing a better understanding of digital tools and platforms. It also enhanced awareness about online communication, safety, and responsible use of technology.
